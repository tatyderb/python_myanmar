### Зачем читать данные

Мы уже написали программу, которая находит корни уравнения $x^2 + x - 20 = 0$. 

```python
# коэффициенты уравнения
a = 1
b = 1
c = -20

# дискриминант
d = (b**2 - 4*a*c)**0.5

# корни
x1 = (-b + d)/(2*a)
x2 = (-b - d)/(2*a)

# печать ответа
print(x1, x2)    # напечатает 4.0 -5.0
```

Если заменить часть, где записываем конкретные коэффициенты уравнения в переменные `a`, `b`, `c` на **чтение чисел с клавиатуры**, то получим программу, которая может решать любые квадратные уравнения.

*У которых есть вещественные корни. Как написать, что "корней нет", если дискриминант отрицательный, мы объясним в модуле про условные операторы.*


### Чтение данных

Python умеет читать. Вы вводите с клавиатуры. Python читает с помощью функции **input()**.

* Программа печатает с помощью `print()`. Это называется **вывод программы** (output).
* Программа читает с помощью `input()`. Это называется **ввод программы** (input).

Программа читает ваше имя, сохраняет его в переменную `name` и здоровается.

```python
print('Как тебя зовут?')
name = input()
print('Здравствуй,', name)
```

| Ввод или вывод? | Код | В консоли |
|----|----|----|
| вывод | `print('Как тебя зовут?')` | `Как тебя зовут?` |
| ввод | `name = input()` | `Таня` |
| вывод | `print('Здравствуй,', name)')` | `Здравствуй, Таня` |

Функция **int(строка)** делает из строки число. Если сможет. Из строки `'123'` можно сделать число `123`. Из строки `'Hello'` число сделать нельзя.

![lamp](https://stepik.org/media/attachments/lesson/1140054/lamp.gif)
 Название `int()` - это сокращение от английского слова integer (целое число).
 
Показанный выше принцип работает всегда. `type` возвращает тип объекта. Название типа объекта дает функцию преобразования к этому типу (создания объекта этого типа). То есть для числа `1.5` выражение `type(1.5)` вернет `float`. Функция `float('3.14')` из строки `'3.14'` сделает число `3.14`. 


### Прочитаем и сложим два целых числа

Так как нужно прочитать два числа, прочитаем строку (текст) и сделаем из текста целое число.

```python
t = input()     # прочитали строку, сохранили её в переменную t
a = int(t)      # из строки получили число, число сохранили в переменную a
t = input()     # прочитали строку, сохранили её в переменную t
b = int(t)      # из строки получили число, число сохранили в переменную a
res = a + b     # вычислили результат
print(res)      # напечатали результат
```
Можно обойтись без переменных `t` и `res`, подставляя результат выполнения в аргумент следующей функции:
```python
a = int(input())    # прочитали строку, из строки получили число, число сохранили в переменную a
b = int(input())    # прочитали строку, из строки получили число, число сохранили в переменную b
print(a + b)        # вычислили и напечатали сумму
```

### float()

Переделаем программу для сложения вещественных чисел. Их в программировании называют **числа с плавающей точкой** (floating point number). Преобразует к такому числу функция **float()**

```python
a = float(input())    # прочитали строку, из строки получили число, число сохранили в переменную a
b = float(input())    # прочитали строку, из строки получили число, число сохранили в переменную b
print(a + b)          # вычислили и напечатали сумму
```

* ** `float()` может преобразовать '3' в число 3.0, но `int()` не может преобразовать '3.14' в 3.14**

### Преобразования типов

Функции `int()`, `float()`, `str()` преобразуют аргумент к целому, вещественному числу и строке.

```python
a = int('3')        # a = 3
b = int(5.2)        # b = 5
с = float('3.14')   # c = 3.14
d = float(-7)       # d = -7.0
e = str(12)         # e = '12'  строка!!!
f = str(4.5)        # g = '4.5' строка!!!
g = int('hello')    # ошибка
h = float('a 3.14') # ошибка
```

Заметим, что `int('2 3')` тоже приведет к ошибке. Поэтому мы пока читаем входные данные в формате одно число на строку.

## Несколько чисел на одной строке

Чтобы прочитать два целых числа, каждый на своей строке, в переменные `a` и `b` мы пишем так:
```python
a = int(input())
b = int(input())
# вычислим и напечатаем сумму
print(a + b)
```
Входные данные:
```python
2
3
```
### Ошибка 

Если входные данные в виде
```python
2 3
```
То будет ошибка при чтении, то есть при попытке привести строку `'2 3'` к одному целому числу.

```python
x = '2 3'       # x - строка
y = int(x)      # ОШИБКА: из такой строки ОДНО целое число не сделать!
```
получим ошибку при выполнении второй строки:
```python
ValueError: invalid literal for int() with base 10: '2 3'
```
Аналогично при вводе двух чисел, когда мы ожидаем одно число:
```python
t = input()     # t = '2 3'
print(t)        # 2 3
x = int(t)      # ValueError: invalid literal for int() with base 10: '2 3'
```

### Чтение двух целых чисел

Для чтения **нескольких** данных одного типа на одной строке через пробел используют выражение:
```python
a, b = map(int, input().split())    # два целых числа
```
Что делает этот код (можно пока не разбираться в этом, подробнее расскажем позже, пока используем магическое выражение):

* `input()` - прочитать строку
* `input().spit()` - разбить эту строку на слова по пробелам
* `map(int, input().split())` - к каждому слову применить функцию `int()`
* `a, b = map(int, input().split())` - результат записать первое число в `a`, второе число в `b`

Для чтения 3 вещественных чисел нужно изменить количество переменных и функцию `int` заменить на `float`:

```python
x, y, z = map(float, input().split())    # три вещественных числа
```
прочитает строку вида
```python
1.5 -7.345 0.16
```
и запишет в переменные значения `x = 1.5`, `y = -7.345`, `z = 0.16`
